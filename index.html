Try Different Puzzle

    How to play:
  • Set number of tiles in width and height of a board (it is possible, just click on the number) and press button "Make new board" (bootom/left).
  • Drag and drop Shapes onto the board.
  • Each shape can be put only on empty tiles
  • If you need some place to put some shapes temporarily use sandbox ribbon
  • When you are done with initial level try changing number of rows and columns (up) and lower tile dimensions


  • The goal of the game is to find out is there a solution to a given puzzle!
